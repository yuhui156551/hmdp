# hmdp
## 前后端分离点评项目

### 技术栈

  SpringBoot2、MybatisPlus、Redis、hutool、aspectj、MySQL、lombok、lua脚本、nginx。
  
### 实现
  
  主要实现功能：用户注册登录、上传博客、点赞博客、查询商铺、关注博主、优惠券秒杀等。
