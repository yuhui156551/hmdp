# hmdp
## 前后端分离点评项目

### 技术栈

  SpringBoot2、MybatisPlus、Redis、Redisson、hutool、aspectj、MySQL、Lombok、Lua、Nginx。
  
### 实现
  
  主要实现功能：用户注册登录、上传博客、点赞博客、查询商铺、好友关注、优惠券秒杀、附近商户等。
